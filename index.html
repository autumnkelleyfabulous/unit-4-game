Crystal Collector

Let's Play a Game!


  • You will be given a random number at the start of the game.
  • There are four crystals below, each has a different value. By clicking on each crystal you will add the crystal's point value to your total score.
  • To win the game, you must match your total score with the random number shown.If you go over you loose!
  • The value of each crystal is revealed when you click on it.
  • Each time the game begins, the game will change the values of each crystal
